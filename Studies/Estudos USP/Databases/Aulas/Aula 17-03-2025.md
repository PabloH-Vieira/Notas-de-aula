Outros tipos de comandos incluem:

```Plain
Escolha (codigo)
	Caso 1: escreva ("abc");
	Caso 2: escreva ("abc");
FimEscolha	
```

  

No comando enquanto existe a variável contador, a qual é utilizada para controlar quantas vezes o bloco de comandos vai se repetir.

  

A linguagem C é uma linguagem de nível médio, entre suas características estão:

- Não é tipada, permitindo grande flexibilidade na conversão de tipos das variáveis;

- Uso intensivo de ponteiros;

- Definição por blocos através de {}

- Não define operações de entrada e saída, estas são importadas a partir da biblioteca stdio.h
    
    - printf
    
    - scanf
