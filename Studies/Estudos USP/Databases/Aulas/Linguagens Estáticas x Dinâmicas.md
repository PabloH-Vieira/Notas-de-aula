Linguagens estáticas são compiladas e durante o tempo de compilação a estrutura do código é executada, sem haver flexibilidade para alterar o código durante a execução.

- C++

- Java

- Devemos definir as variáveis antes de usá-las.

- Devemos definir tipos das variáveis e dos argumentos dos métodos.

- Checagem dos tipos ocorre em tempo de compilação.

  

Linguagens dinâmicas, como Ruby, Python, Javascript, Lua etc. tem como algumas de suas características:

- Não é preciso declarar as variáveis antes de usá-las.

- Variáveis não têm tipos fixos.

- Checagem dos tipos ocorre em tempo de execução.

- Capacidade de executar trechos da linguagem fornecidos em tempo de execução (por exemplo, é possível executar um código que o usuário tenha escrito no programa).

- Alterar a estrutura do código em tempo de execução.

- Gestão automática da memória.

  

Esses tipos de linguagens têm incorporado algumas características dos outros tipos.
