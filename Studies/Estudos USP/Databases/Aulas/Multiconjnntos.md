Conjuntos são uma coleção de objetos, levando em conta que cada objeto só pode aparecer uma única vez.

Multiconjuntos são uma coleção de objetos, mas cada objeto pode aparecer mais de uma vez. Assim como nos conjuntos, a ordem não importa.

A cardinalidade de um multiconjunto é a quantidade de elementos dele, incluindo as repetições.

Combinações simples correspondem aos subconjuntos de _k_ elementos possíveis dentro de um conjunto com _n_ elementos.

Combinações com repetição correspondem aos multiconjuntos de _k_ elementos escolhidos em um conjunto de _n_ elementos.

Um conceito que ocorre na análise de problemas de conjuntos é o de potência fatorial crescente, denotada por $n^{\overline{k}}$. Consiste no produto de _n_ até (_n+k+1)_ elementos.

  

Multiconjuntos dizem respeito às maneiras que existem de organizar um conjunto de _n_ itens. A cardinalidade de um subconjunto é a soma de todos os elementos, incluindo as repetições. Chamamos de multiplicidade a quantidade de vezes que um elemento se repete, e a soma das multiplicidades é igual à cardinalidade, obviamente.
